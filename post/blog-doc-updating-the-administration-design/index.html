    Blog-Doc, updating the administration design!

    Posted: 2023-04-22 | Updated: 2025-06-03
    Category: web-development
    Tagged: blog-doc admin-ui tabler cms design

    Blog-Doc gets a redesigned admin interface using Tabler for a modern and user-friendly content management experience.

    Intro

    I've build the first version of the administration using Pure.css because Blog-Doc uses it.
    Since I wanted to split the menu out of the settings and give it a dedicated page, I told myself:
    "Why not a whole new design?!"

    The new design

    Some time ago, I had to build a project with a modern administration interface and used Tabler for that.
    You guessed it, Blog-Doc administration now uses Tabler!
