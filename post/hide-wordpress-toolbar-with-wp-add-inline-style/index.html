    Hide WordPress Toolbar with wp_add_inline_style

    Posted: 2020-05-22 | Updated: 2025-06-03
    Category: wordpress
    Tagged: wordpress wp-add-inline-style admin-bar toolbar custom-css

    Learn how to hide the WordPress admin toolbar using wp_add_inline_style - a clean, code-based approach for theme customization.

    Ever wanted to hide the WordPress Toolbar?
    If you are a new developer, you should know that it's not allowed to hide it directly wit CSS.

    The WordPress Toolbar is the little black bar on top of the site that only the logged in user(s) can see, depending on their role(s) and how the Super Admin or the Administrator has configured the visibility of this bar for each one‚Ä¶

    We can set the display status of this bar for the front side of our website by using the show_admin_bar() function.

    Also, we can directly target it by it‚Äôs CSS id #wpadminbar and hide it like so:

    It‚Äôs not the main reason, you can have any other reason to hide it with CSS without using a stylesheet, scenarios are limitless.

    This is where wp_add_inline_style() function steps in üôÇ
    In your functions.php file, add the following: 

    /**
 * Hide the Toolbar using inline style since we can't use #wpadminbar in CSS.

        #wpadminbar{
            display: none;
        }";
    wp_add_inline_style( 'my-stylesheet-handle', $hide_adminbar );
}
add_action( 'wp_enqueue_scripts', 'myfunction_hide_adminbar' );

    You should replace it with the $handle of your stylesheet!
