    Old Writer Theme

    Old Writer is a high-contrast, accessibility-first blog theme for Aether CMS that blends modern web design with classic computer style. Built using plain HTML, CSS, and JavaScript, it ensures excellent readability and works smoothly on all devices.

    Design Philosophy

    Old Writer proves that accessible design can also look great. It uses strong contrasts to keep text easy to read in bright daylight or in dark rooms. You can choose between two modes: black text on white for regular reading, or green text on black for a classic computer look.

    Uncompromising Accessibility

    • Complete keyboard navigation support

    • Semantic HTML structure throughout

    • Auto-generating table of contents for long-form articles

    • Smooth scroll navigation with progress tracking
    • Smart heading hierarchy detection (H2, H3, H4)
